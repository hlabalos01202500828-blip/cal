📘 Algebra Fundamentals

Before calculus, you need to be solid with these. No skipping! 🚫

Exponent Rules

x^a · x^b = x^(a+b)
(x^a)^b = x^(ab)
x^(-n) = 1/x^n
x^(1/n) = ⁿ√x    x^(m/n) = ⁿ√(x^m)
x^0 = 1 (x ≠ 0)

Factoring Techniques

Difference of Squares:
a² - b² = (a+b)(a-b)
Perfect Square Trinomial:
a² + 2ab + b² = (a+b)²    a² - 2ab + b² = (a-b)²
Sum/Difference of Cubes:
a³ + b³ = (a+b)(a² - ab + b²)
a³ - b³ = (a-b)(a² + ab + b²)
Grouping / AC Method:
ax² + bx + c → find two numbers that multiply to ac and add to b

Rational Expressions

a/b · c/d = ac/bd
a/b ÷ c/d = a/b · d/c = ad/bc
a/b + c/d = (ad + bc)/bd
To simplify a complex fraction: multiply numerator AND denominator by the LCD of all smaller fractions inside.

Rationalizing

To eliminate a radical from the denominator, multiply by its conjugate:
1/(√a + √b) × (√a - √b)/(√a - √b) = (√a - √b)/(a - b)
This is HUGE for limit problems! Whenever you see 0/0 and there's a radical, try rationalizing.

Logarithm Rules

log(ab) = log a + log b
log(a/b) = log a - log b
log(a^n) = n·log a
log_b(b) = 1    log_b(1) = 0
ln(e^x) = x    e^(ln x) = x

🎯 Limits — The Big Picture

A limit describes the value a function approaches as the input approaches some value — even if the function is never defined there!
lim[x→a] f(x) = L
This says: as x gets closer and closer to a, f(x) gets closer and closer to L.
Limits are about the journey, NOT the destination. f(a) might not even exist but the limit can still exist!

One-Sided Limits

lim[x→a⁻] f(x) — Left-hand limit (x approaches a from the LEFT)
lim[x→a⁺] f(x) — Right-hand limit (x approaches a from the RIGHT)
lim[x→a] f(x) exists ⟺ lim[x→a⁻] f(x) = lim[x→a⁺] f(x)
If left limit ≠ right limit, the two-sided limit does NOT exist (DNE)!

Methods for Evaluating Limits

1. Direct Substitution — just plug in x = a. Works when f is continuous at a.
lim[x→2] (x² + 3x) = 4 + 6 = 10 ✓
2. Factor & Cancel — for 0/0 indeterminate form with polynomials.
lim[x→3] (x²-9)/(x-3) = lim (x+3)(x-3)/(x-3) = lim (x+3) = 6
3. Rationalize — multiply by conjugate when radicals cause 0/0.
lim[x→0] (√(x+1)-1)/x · (√(x+1)+1)/(√(x+1)+1) = lim x/(x(√(x+1)+1)) = 1/2
4. Trig Identities — rewrite using sin/cos identities. 5. L'Hôpital's Rule — for 0/0 or ∞/∞: take derivative of top and bottom separately!
lim f(x)/g(x) [0/0 form] = lim f'(x)/g'(x)

Special Trig Limits (MEMORIZE!)

lim[x→0] sin(x)/x = 1
lim[x→0] (1 - cos x)/x = 0
lim[x→0] tan(x)/x = 1
lim[x→0] (1 - cos x)/x² = 1/2
These are used ALL the time. When you see sin(something)/something, you can bet the answer is 1!

Limits at Infinity

Polynomial/Rational Functions — look at leading terms only!
lim[x→∞] (3x² + 5x) / (2x² - 1) = 3/2 (same degree → ratio of leading coefficients)
lim[x→∞] (5x + 1) / (x² - 3) = 0 (numerator degree < denominator)
lim[x→∞] (x³ + 1) / (2x + 1) = ∞ (numerator degree > denominator)
Horizontal Asymptote: y = L if lim[x→±∞] f(x) = L
lim[x→∞] 1/x = 0    lim[x→∞] 1/x^n = 0 (n > 0)

Infinite Limits (Vertical Asymptotes)

lim[x→0⁺] 1/x = +∞    lim[x→0⁻] 1/x = -∞
Occurs when denominator → 0 but numerator doesn't. Check sign from each side!
If f(x) → ∞ from both sides: the limit is ∞ (DNE). If ∞ from one side and -∞ from the other: limit is DNE.

The Squeeze Theorem

If g(x) ≤ f(x) ≤ h(x) near a, and lim g(x) = lim h(x) = L, then lim f(x) = L.
Classic: -1 ≤ sin(x) ≤ 1 so -x² ≤ x²sin(1/x) ≤ x²
⟹ lim[x→0] x²sin(1/x) = 0

🔥 Derivatives — The Core of Calculus

Definition of the Derivative

The derivative is the instantaneous rate of change (slope of the tangent line):
f'(x) = lim[h→0] [f(x+h) - f(x)] / h
Also written as: dy/dx, Df(x), ẏ, y'
The derivative tells you how fast the function is changing at any point. Positive → increasing, Negative → decreasing, Zero → flat (possibly a max or min).

Basic Differentiation Rules

d/dx [c] = 0 (constant rule)
d/dx [x^n] = nx^(n-1) (power rule)
d/dx [cf(x)] = c·f'(x) (constant multiple)
d/dx [f ± g] = f' ± g' (sum/difference)

Product Rule

d/dx [f·g] = f'g + fg'
Memory: "first times derivative of second, plus second times derivative of first"
Example: d/dx [x²·sin x] = 2x·sin x + x²·cos x

Quotient Rule

d/dx [f/g] = (f'g - fg') / g²
Memory: "low d-high minus high d-low, over low squared below"
Example: d/dx [x²/cos x] = (2x·cos x - x²·(-sin x)) / cos²x
Don't forget the MINUS sign! And it's always over g², not g.

Chain Rule

For composite functions f(g(x)):
d/dx [f(g(x))] = f'(g(x)) · g'(x)
Memory: "derivative of outside (leaving inside alone) TIMES derivative of inside"
Example: d/dx [sin(x²)] = cos(x²) · 2x = 2x·cos(x²)
Example: d/dx [(3x+1)^5] = 5(3x+1)^4 · 3 = 15(3x+1)^4
Chain rule is used when you have a function INSIDE another function. Always ask: is there an inner function?

Derivatives of Trig Functions

d/dx [sin x] = cos x     d/dx [cos x] = -sin x
d/dx [tan x] = sec²x    d/dx [cot x] = -csc²x
d/dx [sec x] = sec x·tan x    d/dx [csc x] = -csc x·cot x
The "co-" functions (cos, cot, csc) always have a NEGATIVE derivative. Remember that!

Derivatives of Exponential & Log Functions

d/dx [eˣ] = eˣ    d/dx [aˣ] = aˣ·ln(a)
d/dx [ln x] = 1/x    d/dx [log_a x] = 1/(x·ln a)
eˣ is its own derivative — the only function like that in math! Incredibly special.

Implicit Differentiation

When y is not isolated, differentiate both sides w.r.t. x, treating y as a function of x:
d/dx [y²] = 2y · dy/dx (chain rule — y is inside)
Example: x² + y² = 25 → 2x + 2y(dy/dx) = 0 → dy/dx = -x/y
Every time you differentiate a y-term, multiply by dy/dx (chain rule). Then solve for dy/dx at the end.

Higher Order Derivatives

f''(x) = d²y/dx² = derivative of f'(x)
f'''(x) = d³y/dx³ = derivative of f''(x)
Physical meaning: f'(x) = velocity, f''(x) = acceleration (if f is position).

〰️ Concavity & Inflection Points

What is Concavity?

Concavity describes whether a curve "cups upward" or "cups downward."
Concave UP (∪): f''(x) > 0 — like a bowl, can hold water
Concave DOWN (∩): f''(x) < 0 — like an upside-down bowl
If you're on a concave up section of a road, you'd feel pressed into your seat (accelerating up). Concave down = you'd float out of your seat.

Second Derivative Test for Concavity

Steps:
1. Find f''(x)
2. Find where f''(x) = 0 or f''(x) is undefined
3. Test intervals using a sign chart for f''(x)
4. Where f''(x) > 0 → concave up; f''(x) < 0 → concave down

Inflection Points

An inflection point is where the concavity CHANGES (from up to down or vice versa).
Necessary condition: f''(x) = 0 or f''(x) DNE at that point
But f''(c) = 0 does NOT guarantee an inflection point — the sign must actually CHANGE!
Inflection points are like the transition between an uphill curve and a downhill curve on a rollercoaster. The "S-curve" shape has an inflection point in the middle.

How Concavity Relates to Derivatives

f'(x) increasing ↔ f''(x) > 0 ↔ Concave up
f'(x) decreasing ↔ f''(x) < 0 ↔ Concave down
This means: on a concave up section, the tangent line slopes are getting steeper (more positive or less negative).

Complete Curve Sketching Strategy

Step 1: Domain (where is f defined?)
Step 2: Intercepts (f(0) and f(x)=0)
Step 3: Find f'(x) → critical points (where f'=0 or DNE)
Step 4: Sign chart of f'(x) → intervals increasing/decreasing
Step 5: Find f''(x) → find where f''=0
Step 6: Sign chart of f''(x) → concavity + inflection points
Step 7: Asymptotes (lim as x→∞ or at discontinuities)
Step 8: Put it all together and sketch!

⛰️ Maxima & Minima

Definitions

Local (Relative) Maximum: f(c) ≥ f(x) for all x near c
Local (Relative) Minimum: f(c) ≤ f(x) for all x near c
Absolute (Global) Max/Min: the largest/smallest value over the entire domain
Local max/min are like hilltops and valley bottoms along a hike. The absolute max is the highest peak, absolute min is the lowest valley.

Critical Points

A point c is a critical point if f'(c) = 0 OR f'(c) is undefined.
ALL local extrema occur at critical points, but NOT all critical points are local extrema!
Example: f(x) = x³ has f'(0) = 0, but x=0 is NOT a local extremum — it's an inflection point!

First Derivative Test

At a critical point c:
• If f' changes from + to −: LOCAL MAXIMUM
• If f' changes from − to +: LOCAL MINIMUM
• If f' doesn't change sign: NEITHER (inflection point)
Make a sign chart for f'(x)! Test a number in each interval around the critical points.

Second Derivative Test

At a critical point c where f'(c) = 0:
If f''(c) > 0 → Local MINIMUM (concave up = valley)
If f''(c) < 0 → Local MAXIMUM (concave down = hill)
If f''(c) = 0 → INCONCLUSIVE (use first derivative test instead)

Absolute Extrema on a Closed Interval [a,b]

Closed Interval Method (Extreme Value Theorem):
1. Find all critical points in (a, b)
2. Evaluate f at all critical points and at endpoints a and b
3. The largest value is the absolute max, smallest is the absolute min
Don't forget the endpoints! Many students find all the critical points but forget to check f(a) and f(b).

Optimization Problems

Strategy:
1. Read carefully. Draw a diagram if possible.
2. Identify what you want to MAXIMIZE or MINIMIZE.
3. Write the objective function (what you're optimizing).
4. Use constraint equations to reduce to ONE variable.
5. Take derivative, set equal to 0, solve.
6. Verify it's actually a max/min using first or second derivative test.
7. Answer the question (with units!).

⏱️ Related Rates

What are Related Rates?

Related rates problems ask: if one quantity is changing at a known rate, how fast is a related quantity changing?
Key tool: Implicit differentiation with respect to TIME (t)
Everything is changing with respect to time! dV/dt means "how fast is Volume changing per unit time".

General Strategy (MEMORIZE THIS PROCESS!)

Step 1: DRAW a diagram and label everything with variables.
Step 2: IDENTIFY what rates are given (dX/dt = ?) and what rate you need (dY/dt = ?).
Step 3: WRITE an equation relating all relevant variables.
Step 4: DIFFERENTIATE both sides with respect to t (chain rule on everything!).
Step 5: SUBSTITUTE known values and SOLVE for the unknown rate.
Substitute the given values of the variables AFTER differentiating, not before!

Common Geometric Formulas (You Need These!)

Circle: A = πr²    C = 2πr
Rectangle: A = lw    P = 2l + 2w
Triangle: A = ½bh
Sphere: V = (4/3)πr³    SA = 4πr²
Cylinder: V = πr²h    Lateral SA = 2πrh
Cone: V = (1/3)πr²h
Pythagorean: a² + b² = c²

Key Related Rates Templates

Expanding Circle: A = πr² → dA/dt = 2πr · dr/dt
Expanding Sphere: V = (4/3)πr³ → dV/dt = 4πr² · dr/dt
Ladder Problem: x² + y² = L² → 2x(dx/dt) + 2y(dy/dt) = 0
Shadow Problem: Similar triangles + implicit differentiation
Trough/Tank filling: Volume formula → differentiate w.r.t. t
Angle problems: tan θ = y/x → sec²θ · dθ/dt = (x·dy/dt - y·dx/dt)/x²
